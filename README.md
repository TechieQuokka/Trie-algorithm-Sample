# Trie Algorithm Sample

C#으로 구현한 Trie(트라이) 자료구조 샘플 코드입니다.

## 프로젝트 구조

```
├── Trie/                    # Trie 자료구조 구현
├── Trie algorithm Test/     # 테스트 코드
└── Trie algorithm Test.sln  # 솔루션 파일
```

## Trie 자료구조

- 문자열 검색에 최적화된 트리 자료구조
- 자동완성, 사전 구현 등에 활용

## 요구사항

- .NET Framework / .NET Core
- Visual Studio 2019+

## 빌드 및 실행

1. `Trie algorithm Test.sln` 파일을 Visual Studio에서 엽니다
2. 솔루션을 빌드합니다
3. 테스트를 실행합니다
